各家网盘对比

Dec 25, 2014 | Web | Hits

Dropbox

  • 确实是首选，服务最好，没有之一。
  • 唯一的缺点是已经GFW，国内同步太痛苦，习惯了后台同步再去适应每次都要记得连接VPN然后同步完在关电脑，确实是一件很难的事情。
  • 舍弃

box

  • 没注册成功过。。

微软家

  • 同样被GFW

百度网盘

  • 国内的网盘就用过百度一家，还买了一年的VIP。
  • 容量确实大，单个文件最大可以是20G，存电影必备
  • 舍弃的原因是开了一个网盘存小电影竟然全部和谐了。而且是在我不知道的情况下。完全不能忍
  • 另外一个原因是国内网盘的不耻行为，只要上传到网盘，意味这个文件你永远都删不了。秒传功能，就是这样来的。
  • 不付费，我们就是产品，而国内的厂家，不管你付费不付费，你就是产品
  • 坚决舍弃。

seafile

  • 自建网盘，开源。
  • 如果用他们家的服务器，有2G空间，自己有vps的话，就无所谓了。
  • 配置很简单，官网上都有
  • 关键 osx iOS windows wp8.1 andoid 树莓，全有客户端。开源的好处显示出来了。
  • 坚决支持，目前我的解决方案 seafile

大致如此。
