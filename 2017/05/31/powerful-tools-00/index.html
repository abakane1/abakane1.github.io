小利器00--耳机

May 31, 2017 | Power tools | Hits

缘起: 随着年龄上升，越来越感觉如无必要，务增实体的重要性。这个系列就记录下每一个物件的去留。
利器借用一直订阅的一个blog的说法。个人的话，只能说小利器。

耳机需求

  1. 舒服感：需要可以长期的佩戴不难受。
  2. 无线化：地铁出行的时候拖着一个长长的线总会某名的挂到某些地方。
  3. 音质：木耳，听个大概。没有深入研究这块。

用过的耳机们

  1. AKG K420 便携头戴式耳机(海军蓝)-数码影音-亚马逊中国 [akg420] 经典K420 通过zhihu推荐。

    1. 音质确实比一般的耳机出色很多很多
2. 开放式的耳机，在安静的地方能听到很清楚的歌曲声音。
3. 夏天热的时候吸汗，运动的时候能难受
4. 夹耳朵，带着眼镜带着时间久了很疼。
5. 性价比还是很不错的。

  2. 苹果小白

    1. 基本人手一个，音质也说的过去。
2. 但是配件好多都是假的，买的水货手机很容易拿到假的。
3. 运动带不住，可能不适合我的耳朵，健身还行，跑步就完蛋。
4. 有线，很容易扯坏，耐脏，偶尔还能打电话。

  3. Bose SoundSport In-Ear Headphones Black SoundSport IE HP BLK - Best Buy

    1. 便宜的运动型耳机，动辄上千的bose黑五的时候200RMB拿下。
2. 带耳翼，跑步完全没有问题。
3. 佩戴舒服，音质也就听个响。
4. 无麦，有线是比较大的缺点。
5. 存在所有入耳耳机的问题，有回响。听久了脑袋嗡嗡叫。
6. 停产了，后期好像有一款无线的。低价可以入。

  4. Beats Powerbeats3 Wireless Earphones - Beats by Dre

    1. 目前服役款，黑色。红色造型最拉风，地铁里面随时能注意到。
2. W1芯片是亮点
  - 我手头上有IPAD, IWATCH, IPHONE 和MACBOOK PRO。
  - 有了W1芯片之后可以无缝链接，比如我在Iphone上用PB3听歌，
  - 突然想在Pro上看个视频，直接蓝牙链接
  - PB3可以自动断开Iphone，而不需要多次匹配。
3. 无线，可以挂在脖子上，完全不怕丢。
4. 轻度防水，运动的时候完全无感。
5. 充满电可以持续播放大概8个小时。
6. 唯一不足的地方可能就是入耳的缘故，带久了还是会难受。
7. 稍贵，我买的150刀，现在好像税前120就能拿下。

Wishlist

  1. Bose QC 35
    1. 主动降噪，带上之后进入一个自己的世界。旁边的人根本听不见。
2. 包耳，长时间佩戴也感觉也不错。
3. 夏天可能会热一些。
4. 音质也就那样。
5. 价格是阻止一切的因素。
